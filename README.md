# Game-Recommendation-Web-App
The Game Recommendation Web App allows users to search their favorite games and discover similar games they might enjoy.

The frontend was deployed using Vercel, at: 'https://game-recommendation-web-a5lmni746-ryanfelldowns-projects.vercel.app/'.

The backend was deployed using Render, at: 'https://game-recommendation-flask-backend.onrender.com/api/process-input/'.
